# 🌀 Velopath Backend

Backend API for **Cyclint Smart Bicycle Navigation System**, built using:
- Node.js + Express
- PostgreSQL + PostGIS
- Flutter frontend integration

## 🚀 Features
- Geospatial data storage and queries
- Routing API for personalized paths
- Integration with Flutter app
- RESTful endpoints
